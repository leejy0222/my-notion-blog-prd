**2-2: TypeScript 타입**

- `PostCard.tsx` - 글 카드
- `PostContent.tsx` - Notion 블록 렌더링

**2-4: 레이아웃**
- `app/layout.tsx` - 루트 레이아웃 (Header + Footer)

**2-5: 유틸리티 함수**
- `formatDate(date)` - 날짜 포맷팅

## 🛠️ 기술 스택

### Frontend
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 3 + shadcn/ui
- **Forms**: React Hook Form + Zod
- **State Management**: Zustand
- **Icons**: Lucide React
- **Theme**: next-themes

### CMS & Backend
- **CMS**: Notion API
- **API Client**: @notionhq/client
- **Caching**: Next.js ISR

### Deployment
- **Hosting**: Vercel
- **Version Control**: GitHub
- **CI/CD**: GitHub Actions (Phase 5)
